kali 工具文档

hydra

Hydra 是一个并行的登录破解器，支持众多协议的攻击。它非常快速和灵活，而且新的模块很容易添加。

这个工具使研究人员和安全顾问有可能展示远程获得未经授权的系统访问是多么容易。

它支持：Cisco AAA, Cisco auth, Cisco enable, CVS, FTP, HTTP(S)-FORM-GET, HTTP(S)-FORM-POST, HTTP(S)-GET, HTTP(S)-HEAD, HTTP-Proxy, ICQ, IMAP, IRC, LDAP, MS-SQL, MySQL, NNTP, Oracle Listener, Oracle SID、PC-Anywhere, PC-NFS, POP3, PostgreSQL, RDP, Rexec, Rlogin, Rsh, SIP, SMB(NT), SMTP, SMTP Enum, SNMP v1+v2+v3, SOCKS5, SSH（v1 和 v2）, SSHKEY, Subversion, Teamspeak（TS2）, Telnet, VMware-Auth, VNC 和 XMPP。
sudo apt install hydra

nmap

ncat

ncat 是 NMAP 项目对 Netcat 的重新实现，提供了原始实现中的大部分功能，以及一些新的功能，如 IPv6 和 SSL 支持。端口扫描支持已被删除。

nmap

Nmap 是一个用于网络探索或安全审计的工具。它支持 ping 扫描(确定哪些主机在运行)，许多端口扫描技术，版本检测(确定端口后面监听的服务协议和应用程序版本)，和 TCP/IP 指纹(远程主机操作系统或设备识别)。Nmap 还提供灵活的目标和端口规范，诱饵/隐蔽扫描，sunRPC 扫描等等。大多数 Unix 和 Windows 平台在 GUI 和命令行模式下都被支持。几个流行的手持设备也被支持，包括 Sharp Zaurus 和 iPAQ。

hashcat

Hashcat 支持超过 300 种高度优化的散列算法的五种独特攻击模式。Hashcat 目前支持 Linux 上的 CPU、GPU 和其他硬件加速器，并拥有帮助实现分布式密码破解的设施。

hashcat 支持的散列算法的例子有：MD5, HMAC-MD5, SHA1, HMAC-SHA1, MySQL323, MySQL4.1/MySQL5, phpass, MD5(Wordpress), MD5(phpBB3), MD5(Joomla), md5crypt, MD5(Unix), FreeBSD MD5, Cisco-IOS, MD4, NTLM, Domain Cached Credentials（DCC）、MS Cache, SHA256, HMAC-SHA256, md5apr1, MD5(APR), Apache MD5, SHA512, HMAC-SHA512, Cisco-PIX, Cisco-ASA, WPA/WPA2, Double MD5, bcrypt、Blowfish(OpenBSD), MD5(Sun), Double SHA1, SHA-3(Keccak), Half MD5, Password Safe SHA-256, IKE-PSK MD5, IKE-PSK SHA1, NetNTLMv1-VANILLA/NetNTLMv1-ESS、NetNTLMv2, Cisco-IOS SHA256, Android PIN, AIX {smd5}, AIX {ssha256}, AIX {ssha512}, AIX {ssha1}, GOST, GOST R 34, Fortigate（FortiOS）, OS X v10.8+, GRUB 2, IPMI2, RAKP, HMAC-SHA1, sha256crypt, SHA256(Unix), Drupal7, WBB3, scrypt, Cisco $8$, Cisco $9$, Radmin2, Django（PBKDF2-SHA256）, Cram MD5, SAP, iSSHA-1、PrestaShop, PostgreSQL, Challenge-Response Authentication (MD5), MySQL Challenge-Response, Authentication (SHA1), SIP digest authentication (MD5), Plaintext, Joomla < 2.5.18, PostgreSQL, osCommerce, xt:Commerce, Skype, nsldap, Netscape, LDAP, nsldaps, SSHA-1（Base64）, Oracle S：Type (Oracle 11+), SMF > v1.1, OS X v10.4, v10.5, v10.6, EPi, Django (SHA-1), MSSQL(2000), MSSQL(2005) , PeopleSoft, EPiServer 6.x < v4, hMailServer, SSHA-512(Base64), LDAP {SSHA512}, OS X v10.7, MSSQL(2012 & 2014), vBulletin < v3.8.5, PHPS, vBulletin > v3.8.5, IPB2+, MyBB1.2+, Mediawiki B type, WebEdition CMS, Redmine.
