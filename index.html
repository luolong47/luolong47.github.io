npx 和npm区别

نُشر في 2023-03-16 عُدل في 2023-03-21

npx 和 npm 都是 Node.js 中的命令行工具，用于管理和安装 JavaScript 包。它们的区别如下：

  1. npm 是 Node.js 的官方包管理器，用于安装和管理本地和全局的包。npx 是 npm 5.2+ 版本中包含的一个工具，可以在不全局安装包的情况下运行本地安装的包。
  2. npm 安装的包通常需要全局安装或在项目中安装，然后才能在命令行中运行。而 npx 可以直接运行不需要安装的包，即使这些包未被安装在本地或全局。
  3. npx 会自动查找当前目录下是否存在指定的命令或包，如果存在则直接运行，否则会先安装再运行。 总之，npx 可以让我们更方便地运行临时依赖，而无需手动安装它们。
